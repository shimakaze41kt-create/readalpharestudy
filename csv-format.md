# リードα 復習データ CSVフォーマット

## 普段使うCSV

ファイル: `lead-alpha-template.csv`

| 列名 | 入力例 | 意味 |
| --- | --- | --- |
| 問題番号 | 例題12 | 問題集上の番号。アプリ内のIDになります。 |
| 分野 | 力学 | 力学、波動、熱、電磁気など。 |
| 種類 | 基本例題 | `基礎CHECK`、`基本例題`、`基本問題`、`応用問題` のどれか。 |
| 段階 | 2 | 基礎CHECK=1、基本例題=2、基本問題=3、応用問題=4。 |
| 前提問題 | 基礎C-05,例題12 | この問題を解く前に固めたい問題。複数はカンマ区切り。 |
| 関連問題 | p42-3 | 類題、発展先、同じ考え方を使う問題。複数はカンマ区切り。 |
| 回数 | 3 | これまで解いた回数。 |
| 正解数 | 1 | これまで正解した回数。 |
| 最終実施日 | 2026-05-12 | 最後に解いた日。未着手なら空欄。 |
| メモ | 符号ミス | ミスの原因や次回注意点。 |

## 関係性だけを一覧化するCSV

ファイル: `lead-alpha-relations-format.csv`

| 列名 | 入力例 | 意味 |
| --- | --- | --- |
| 元問題番号 | 例題12 | 関係の出発点になる問題。 |
| 関係 | 前提 | `前提`、`類題`、`発展`、`復習推奨` など。 |
| 先問題番号 | p42-3 | 関係の先にある問題。 |
| 強さ | 強 | 優先度の強さ。`強`、`中`、`弱` など。 |
| メモ | 基本例題から基本問題へ | 関係の理由。 |

アプリに読み込ませるのは普段使うCSVです。関係性だけのCSVは、問題集を整理するときの下書きや一覧表として使う想定です。
